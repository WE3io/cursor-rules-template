# Archive

Historical source material for the AI Blindspots collection.

## Purpose

This directory preserves the original content from Edward Z. Yang's AI Blindspots blog ([ezyang.github.io/ai-blindspots](https://ezyang.github.io/ai-blindspots/)) for reference and content extraction. It is not actively maintained.

## Contents

| Item | Description |
|------|-------------|
| `original-html/` | 20 HTML articles from the original blog (downloaded) |
| `original-rules-v1.0.md` | Synthesized rules from v1.0 (March 2025) |
| `AI Blindspots - A Collection of Insights.pdf` | PDF compilation of insights |
| `extract_content.sh` | Script for extracting content from HTML |

## Retention

- **Keep for reference** — Used when enhancing articles or rules from the canonical collection.
- **No active changes** — Content is historical; updates go to `../articles/` and `../rules/` instead.
- **Links** — Articles may reference `original-html/` for the original 20 topics.
