## How to Use

### Direct Integration
```bash
# Copy to your project
cp ai-coding-assistant-rules.md /your-project/.cursor/rules
```

### Selective Integration
Pick relevant sections for your `.cursor/rules`:
```markdown
## Security Boundaries
[Copy from Security-Specific Practices]

## Our Project
- File size limit: 64KB
- Testing: Black box only
- Type checking: Strict mode

## Common Patterns
[Adapt from Implementation Guidelines]
```

### Reference Document
Keep it as a reference:
- Code reviews: Security checklist
- Debugging: Scientific debugging method
- Stuck: "When to Restart Session" decision tree
